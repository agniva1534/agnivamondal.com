# Assets Folder

This folder contains static assets for the portfolio website.

## Resume File

Please add your resume PDF file here with the name: **`Agniva_Mondal_Resume.pdf`**

### Instructions:

1. Save your resume as a PDF file
2. Name it exactly: `Agniva_Mondal_Resume.pdf`
3. Place it in this `assets` folder
4. The download buttons on the website will automatically work!

### Alternative:

If you want to use a different filename, update the file paths in `index.html`:
- Search for `assets/Agniva_Mondal_Resume.pdf` and replace with your filename

### File Format:

- **Format**: PDF (recommended)
- **File Size**: Keep under 5MB for best performance
- **Naming**: Use underscores or hyphens, avoid spaces
