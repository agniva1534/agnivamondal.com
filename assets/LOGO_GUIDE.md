# Logo Guide

This folder contains all logo variations for Agniva Mondal's portfolio website.

## Logo Files

### 1. `logo.svg` - Full Logo (Recommended for Navigation)
- **Size**: 200x60px
- **Use**: Navigation bar, headers
- **Features**: Icon + Full name + Tagline
- **Best for**: Desktop navigation, main branding

### 2. `logo-icon.svg` - Icon Only
- **Size**: 60x60px
- **Use**: Mobile navigation, favicon, social media
- **Features**: Circular icon with data chart and healthcare cross
- **Best for**: Compact spaces, mobile view

### 3. `logo-text-only.svg` - Text Only
- **Size**: 180x40px
- **Use**: Email signatures, documents
- **Features**: Name with gradient text
- **Best for**: Professional documents, email headers

### 4. `favicon.svg` - Browser Favicon
- **Size**: 32x32px
- **Use**: Browser tab icon
- **Features**: Simplified icon version
- **Best for**: Browser favicon

## Logo Design Elements

### Colors
- **Primary**: Teal (#0d9488, #14b8a6)
- **Secondary**: Slate/Dark (#0f172a)
- **Gradient**: Teal to Dark gradient

### Symbolism
- **Data Chart Bars**: Represent data analytics and insights
- **Healthcare Cross**: Represents healthcare and pharma focus
- **Circular Design**: Modern, professional, complete
- **Gradient**: Dynamic, forward-thinking

## Usage Guidelines

### Do's ✅
- Use the full logo in navigation bars
- Use icon version for mobile/small spaces
- Maintain aspect ratio when resizing
- Use on light backgrounds (white, light gray)
- Keep minimum clear space around logo

### Don'ts ❌
- Don't stretch or distort the logo
- Don't change the colors
- Don't place on busy backgrounds
- Don't make the logo too small (minimum 40px height)

## Customization

All logos are SVG files, so they can be easily edited:
- Open in any vector editor (Illustrator, Inkscape, Figma)
- Modify colors in the `<defs>` section
- Adjust sizes by changing viewBox dimensions
- Export to PNG if needed for specific use cases

## File Formats

- **SVG**: Scalable, best for web (current format)
- **PNG**: If you need raster format, export at 2x resolution
- **PDF**: For print materials

## Integration

The logos are already integrated into:

- ✅ Responsive design (hides text on mobile)
